# PongReinterpreted
A reinterpreted co-op version of Pong, developed during HackITAll Hackathon's GameJam

A reinterpreted co-op version of the classic Pong game, in which the two players must work together in order to 
defeat bosses in levels inspired by retro games, such as Pokemon and Doom; developed via Unity during the 2024 HackITAll Hackathon.
